# Hao Yian - Project Portfolio Page

### Project: GoldenCompass

GoldenCompass is a command-line application for managing internship applications, optimised for
fast typists who prefer a keyboard-driven workflow. It helps users track internships, schedule
interviews, search across their data, and stay on top of deadlines — all from the terminal.

Given below are my contributions to the project.

* **New Feature**: Added the `list` command
  ([PR #35](https://github.com/AY2526S2-CS2113-W10-4/tp/pull/35))
    * What it does: Displays all internships currently in the tracker with index numbers and status.
    * Justification: Users need a quick way to see their entire internship portfolio at a glance.
    * Highlights: Integrates with existing `InternshipList` and uses `toString()` to display status tags. Added defensive checks for null lists and invalid entries.

* **New Feature**: Added the `delete` command
  ([PR #76](https://github.com/AY2526S2-CS2113-W10-4/tp/pull/76))
    * What it does: Removes an internship by its 1-based index. Automatically deletes any associated interview to maintain data consistency.
    * Justification: Users need to remove applications they are no longer pursuing.
    * Highlights: Implements auto-deletion of linked interviews. Includes comprehensive input validation (presence, type, range checks). Added logging and assertions for defensive programming.

* **New Feature**: Added the `delete-interview` command
  ([PR #105](https://github.com/AY2526S2-CS2113-W10-4/tp/pull/105))
    * What it does: Removes an interview from an internship while preserving the internship record.
    * Justification: Users may need to cancel or remove an interview without losing the application entry.
    * Highlights: Searches `InterviewList` by company name to find the interview, then removes it from both `InterviewList` and the internship's reference.

* **New Feature**: Added the `search` command
  ([PR #106](https://github.com/AY2526S2-CS2113-W10-4/tp/pull/106))
    * What it does: Searches for internships by company name (`/c`) and/or job title (`/t`) using case-insensitive substring matching.
    * Justification: Users need to quickly find specific applications without scrolling through the entire list.
    * Highlights: Added `matches()` method in `Internship` class for reusable filtering logic. Supports AND logic for combined searches.

* **Enhancement**: Added `/help` flag support for `delete`, `delete-interview`, and `search` commands
  ([PR #107](https://github.com/AY2526S2-CS2113-W10-4/tp/pull/107))
    * Allows users to type e.g. `delete /help` to see usage information without leaving the application.

* **Enhancement**: Added defensive programming and test coverage for core commands
